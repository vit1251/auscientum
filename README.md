# Auscientum

[![CMake on Linux](https://github.com/vit1251/auscientum/actions/workflows/cmake-linux-platform.yml/badge.svg?branch=main)](https://github.com/vit1251/auscientum/actions/workflows/cmake-linux-platform.yml)

## Overview

Auscientum is an audiobook player application built with wxWidgets for the GUI,
SFML for audio playback, and SQLite for bookmark management.

The application can open and play audiobooks stored in ZIP archives.
